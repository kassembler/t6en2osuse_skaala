# t6en2osuse_skaala
proovitöö
Rakenduse käivitamine käib main.py kaudu. 
Käivitamiseks vajalikud teegid:
1) requests
2) PyQt5
3) pyqtgraph

Programmil on põhifunktsionaalsused olemas. Edasiarendamiseks teeksin andmete visualiseerimist paremaks nii, et kasutades reageeriva disaini omadusi teksid üldse ei kattuks, olenemata akna suurusest ja et joonte pikkus teksti all oleks paremini seotud andme suurusega (kuid siis tekiks mingis mõttes tavaline xy graafik, mitte üks telg). Otsiksin ka huvitavamaid andmeid ja andmekogusid, praegu leidsin ainult TAI ja statistikaameti, kust andmeid API kaudu pärida.
